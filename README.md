# ✨ vue-mo.js ✨

A small wrapper for integrating Mo.js to Vue.js

🙇 Work In Process ... 🙇

## 🏹 Installation

Using npm:

```
$ npm i @azukisiromochi/vue-mo.js
```

Using yarn:

```
$ yarn add @azukisiromochi/vue-mo.js
```

## 🔫 Usage

Using Vue.js:

```javascript
import Vuemo from '@azukisiromochi/vue-mo.js';
Vue.use(Vuemo);
```
